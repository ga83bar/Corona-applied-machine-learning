\doxysubsection{Detailed Description}
Twitch statistics processing file. Acquires raw data, processes it with pandas and saves it into csv files. 

Data is obtained via the requests module from the twitch statistics page, parsed with Beautiful\+Soup and then sorted into a pandas data frame.

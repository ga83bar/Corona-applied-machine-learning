\doxysubsection{Detailed Description}
Script to create work packages for the Docker containers. 

Scrapes all available countries from Socialblade, then also scrapes the respective top 250 channel lists from the site. Reads the user input on how many work packages should be created, splits the link list accordingly and creates the necessary folders.

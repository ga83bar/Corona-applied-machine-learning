\doxysubsection{Detailed Description}
Pipeline to process the gathered data from our sources. 

Scales the data, computes running averages for time series where deemed appropriate and saves the scaled data set into two files. One contains all data prior to the corona pandemic start (which we determined to be at 01.\+01.\+2020), and the other one all data points from during the pandemic.
